mzupan's PEAR Channel Server

Registering the channel: 
pear channel-discover mzupan.github.io/pear
Listing available packages: 
pear remote-list -c mzupan
Installing a package: 
pear install mzupan/package_name
Installing a specific version/stability: 
pear install mzupan/package_name-1.0.0
pear install mzupan/package_name-beta
Receiving updates via a feed: 
http://mzupan.github.io/pear/feed.xml

Protobuf - PHP implementation of Google's Protocol Buffers

Protobuf for PHP is an implementation of Google's Protocol Buffers for the PHP language, supporting its binary data serialization and including a protoc plugin to generate PHP classes from .proto files.
